# HR Analytics Dashboard – Employee Attrition Analysis



### Project Overview



This project showcases an interactive HR Analytics dashboard built using Tableau Public to analyze employee attrition and workforce trends.

The dashboard focuses on visualizing key HR metrics and patterns to support data-driven understanding of employee retention and job satisfaction.



### Objectives



Analyze overall employee attrition and workforce size



Identify departments and job roles with higher attrition



Examine attrition trends across age groups and gender



Visualize job satisfaction patterns across roles

### Dataset Description



HR analytics dataset with structured employee-level data



Includes information on attrition, age, department, job role, gender, education field, and job satisfaction



Data is formatted for analytical visualization and dashboard development



### Tools Used

Microsoft Excel – Basic data inspection

#### Key Insights



The attrition rate reflects moderate employee turnover



Certain departments and job roles show higher attrition levels



Employees in the 25–34 age group experience higher attrition



Lower job satisfaction is associated with increased attrition
